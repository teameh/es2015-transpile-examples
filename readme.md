# ES2015 Transpile Examples

This repo shows how some of the examples from the [Babel Learn ES2015](https://babeljs.io/docs/learn-es2015/) page gets transpiled. 

## Installation
```
npm i
```

## Usage

```
npm start
```

Refresh page + open console to see output

## Development

First shell

```
npm run babel
```

Second shell

```
npm run webpack
```
